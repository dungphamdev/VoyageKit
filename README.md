# Scan & Suggest - Mobile App

A cross-platform mobile application built with React Native and Expo that detects objects using AI and suggests missing items.

## Technologies

- **React Native** (via Expo)
- **TypeScript**
- **Object Detection** (Planned: TensorFlow Lite or Expo Camera with ML Kit)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [Expo Go](https://expo.dev/expo-go) app on your physical device (iOS/Android)
- `npm` or `yarn`

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd antigravity
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

1. Start the development server:
   ```bash
   npm run start
   ```

2. Scan the QR code:
   - **Android**: Use the Expo Go app.
   - **iOS**: Use the Camera app or Expo Go.
